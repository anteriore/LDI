
# Start
1. git pull origin server-branch

# Tech
- Java runtime environment version 1.8.0_162
- Java 8 Update 162
- Node v 6.9.3

# Setup
1. SuperAdmin Folder - `mvn install`
2. app-ui - `npm install`

# How to run
1. run superadmin folder - `mvn spring-boot:run`
2. run app-ui folder - `npm start`
